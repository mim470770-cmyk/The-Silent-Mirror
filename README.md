# The Silent Mirror 
**Theme-w5: Meaning found beyond words.**

This is a C++ Console Application that detects human emotions through visual pattern recognition.

###  Features
- **Abstract Pattern Testing:** No words, just shapes.
- **Dynamic Emotion Logic:** High-sensitivity scoring for accurate results.
- **Emoji Support:** Visual feedback using modern Unicode.
- **Premium UI:** Blue-themed console interface.

###  How to Run
1. Download the `main.cpp` file.
2. Open in VS Code or any C++ Compiler.
3. Run the code and enjoy the experience!
